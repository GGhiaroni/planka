# Sidecars — Tuninho DevOps Env

Cada projeto catalogado tem uma pasta aqui com:

- `env-catalog.json` — Catalogo completo (machine-readable, schema v1.0.0)
- `config.md` — Resumo human-readable do ambiente

O arquivo `server-inventory.json` na raiz consolida TODOS os projetos do servidor.

**IMPORTANTE**: Sidecars sao locais por projeto/servidor. NAO sincronizar via updater.
